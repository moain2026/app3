## القواعد الحاكمة
- **مصدر الحقيقة = APK المُفكَّك.** أي تعارض يُحسَم لصالح Java الأصلي.
- **لا هلوسة كود.** أي غموض → نتوقّف ونسأل.
- كل سطر جديد يلتزم **Clean Code + SOLID**.
- أسماء أعمدة DB الأصلية محفوظة كما هي.
