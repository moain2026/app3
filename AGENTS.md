## 5. الحقائق الحرجة المُتحقَّق منها (احفظها — لا تعيد اكتشافها)

> كل حقيقة هنا مُستخرَجة حرفياً من Java ومُتحقَّق منها. **لا تغيّرها بناءً على
> حدس** — إن شككت، ارجع للملف:سطر المذكور.

| # | الحقيقة | المصدر |
|---|---|---|
| F-1 | `getBaseUrl() = "http://" + IP + ":3000/electric/"` — المنفذ والمسار **ثابتان**، المتغيّر هو IP فقط | `common/AppConfig.java` |
| F-2 | IP الحيّ = `100.87.131.115:3000` عبر **Tailscale VPN فقط** | `AppConfig` / المستخدم |
| F-3 | كل ردود WCF مغلّفة بـ `{اسم العملية}Result` (مثل `LoginResult`, `DeleteReadingResult.Result`) | `*Repository.java` |
| F-4 | تسجيل الدخول `/Login` (L كبيرة)، body `{username, password, appId, secureId}`؛ النجاح = `error_no === 0` | `AuthRepository.java` |
| F-5 | `appId` بصيغة **camelCase** (= رقم الفرع، افتراضي `"1"`). المسار البديل `/UserAuth` بـ `appid` lowercase **غير مستخدم** | `AuthRepository.java` |
| F-6 | `secureId = decimal(أول 8 خانات hex من ANDROID_ID)` — **بلا XOR** (الـ XOR في نظام ترخيص منفصل) | `Defence.java` |
| F-7 | `Delete/UpdateReading → ?appId=&id={num}` حيث `num` = **int** (وليس `noadad` الذي هو String) | `ReadingRepository.java:29,107` |
| F-8 | `Delete/UpdateBond` و`Delete/UpdateBondPayment → id={nmstnd}` | `BondsRepository.java:29,107`, `BondsPaymentRepository.java` |
| F-9 | السند وسند الدفع يستخدمان **نفس الكيان `ItemBonds`** (24 حقلاً) | `entities/ItemBonds.java` |
| F-10 | الصلاحيات = 6 أعلام `int` (`ED,DE,S_K,S_S,REP,SYS`)، المنطق `flag > 0`، و**`SYS > 0` يفتح كل شيء** | `entities/Users.java` |
| F-11 | الطباعة: النص العربي يُرمَّز بـ **Windows-1256 (cp1256)** صراحةً؛ مورد `cp1251` في strings.xml **غير مستخدم** | `printer/bluetooth/PrinterActivity.java:322` |
| F-12 | الطابعة الميدانية الفعلية = **BIXOLON SPP-R310** (ESC/POS عبر Bluetooth SPP). المصنع الأصلي يدعم DPP-250/UNS-SP1B/generic | `PrinterDriverFactory.java` + المستخدم |
| F-13 | محرّك الشبكة الأصلي مزدوج: Retrofit **مُعرَّف لكن غير مستخدم**، و**loopj AsyncHttpClient مُستخدَم فعلياً** في الإنتاج | `model/*Repository.java` |
